# How to Automatically Retry Failed Tests

## Intro

You want reliable tests. But no matter how hard you try, you're going to run into intermittent test failures.
These can be frustrating time wasters that distract you and your team and begin to erode the trust placed in the
automated tests. Once that begins, it's a slippery slope.
